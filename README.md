###--- Library Management System ---###

A full-stack Library Management System built using Node.js, Express, MongoDB & EJS, featuring Admin and User dashboards, book issuing, returns, fines, memberships, and reports.


# Features

# User

  Login & Logout

  View issued books

  View transaction ID

  View fine details

   Membership status


# Admin

  Add books

  Add memberships

  User management

  Issue & return books

  View reports

  View transaction history


 # Reports

    Books report

    Membership report

    Issued books history


#  Tech Stack

  Backend: Node.js, Express.js

  Frontend: EJS, Bootstrap, Custom CSS

  Database: MongoDB

  Authentication: Express Session

  Template Engine: EJS
